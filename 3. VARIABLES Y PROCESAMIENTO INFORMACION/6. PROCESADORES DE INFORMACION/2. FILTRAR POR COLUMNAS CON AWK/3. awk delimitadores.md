Para poner ``delimitadores`` para filtrar utilizaremos el comando **awk**.

Ponemos que las ``,`` van a ser lo que ``delimite las columnas``:

```Bash
echo 'primera columna, segunda columna, tercera columna' | awk -F ',' '{print $NF}'
```

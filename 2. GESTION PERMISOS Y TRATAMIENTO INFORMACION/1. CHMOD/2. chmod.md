**chmod**: para poner o quitar permisos

Para poner permisos de ``usuario``:

```Bash
chmod +x script.sh
```

![[Pasted image 20240613210607.png]]

Para quitar permisos de ``usuario``:

```Bash
chmod -x script.sh
```

![[Pasted image 20240613210702.png]]


``Bucle`` sencillo con ``números``:

```Bash
#!/bin/bash

contador=1

while [ $contador -lt 5 ]; do
        echo "Vuelta número $contador"
        contador=$((contador + 1))
done
```

Le damos permisos:

```Bash
chmod +x script.sh
```


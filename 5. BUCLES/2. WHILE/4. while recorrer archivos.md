
``Bucle`` con ``archivos``:

```Bash
#!/bin/bash

while read -r url; do

        respuesta=$(curl -s -o /dev/null -w "%{http_code}" $url)
		
        if [ $respuesta == "200" ]; then
                echo "La url $url está operativa"
        else
                echo "La url $url no está operativa"
        fi
done < "archivo.txt"
```

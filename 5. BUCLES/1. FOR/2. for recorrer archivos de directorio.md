
``Bucle`` sencillo con ``archivos en directorio``:

```Bash
#!/bin/bash

for i in *
do
        echo "Archivo encontrado: $i"
done
```

![[Pasted image 20240617124132.png]]

Le damos permisos:

```Bash
chmod +x script.sh
```

![[Pasted image 20240615175100.png]]

Ejecutamos el ``script``:

```Bash
./script.sh
```


``Bucle`` sencillo con ``archivos``:

```Bash
#!/bin/bash

for i in $(cat "texto.txt")
do
        echo "Linea encontrada: $i"
done
```

Le damos permisos:

```Bash
chmod +x script.sh
```

![[Pasted image 20240615175100.png]]

Ejecutamos el ``script``:

```Bash
./script.sh
```

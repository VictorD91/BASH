
``Bucle`` sencillo con ``números``:

```Bash
#!/bin/bash

for i in {1..5}
do
        echo "Vuelta número $i"
done
```

![[Pasted image 20240617123449.png]]

Le damos permisos:

```Bash
chmod +x script.sh
```

![[Pasted image 20240615175100.png]]

Ejecutamos el ``script``:

```Bash
./script.sh
```
